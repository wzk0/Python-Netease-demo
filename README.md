# Python-Netease-demo

> 第一个Python小项目——调用了网易云API以获取多个信息

## 依赖:

> 需要安装`requests`，`wget`插件

```
pip3 install requests

pip3 install wget
```

> 需要安装`sox`播放器:

```
pkg install sox
```

## 启动：


```python
python 网易云.py
```

## 参考:

[Node.js版网易云API](https://github.com/Binaryify/NeteaseCloudMusicApi)
